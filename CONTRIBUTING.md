# Contributing to CopyVision

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/cpv.git`
3. Install dependencies: `pnpm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
pnpm dev          # Start Chrome dev mode
pnpm dev:firefox  # Start Firefox dev mode
pnpm test         # Run tests
pnpm lint         # Lint code
pnpm format       # Format code
```

## Pull Requests

- Keep changes focused and minimal
- Add tests for new functionality
- Ensure `pnpm lint` and `pnpm test` pass
- Write clear commit messages

## Reporting Issues

Use GitHub Issues with the provided templates for bug reports and feature requests.
