# Privacy Policy for CopyVision

**Last updated:** March 2026

## Data Collection

CopyVision does **not** collect, store, or transmit any personal data.

## What the extension accesses

- **Images you right-click**: Sent to the Anthropic API for text transcription. Images are not stored by the extension.
- **Anthropic API key**: Stored locally in your browser (`chrome.storage.local`). Never shared with any third party.

## Third-Party Services

CopyVision uses the [Anthropic API](https://www.anthropic.com) to process images. Your use of this API is subject to [Anthropic's privacy policy](https://www.anthropic.com/privacy) and [terms of service](https://www.anthropic.com/terms).

## Data Storage

All data (API key) is stored locally on your device. No data is stored on external servers by CopyVision.

## Changes

We may update this privacy policy. Changes will be posted in this document within the extension repository.

## Contact

For privacy concerns, please open an issue on the [GitHub repository](https://github.com/martinblaha/cpv).
